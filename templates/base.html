Sequence Analysis Sequence Alignment Sequence Variance Sequence Assembly Phylogeny
Analize the number of codons (3-mers), amino acids, or specified k-mers within a genetic sequence. This tool takes in a fna or fastq file containing genetic sequences and analizes the data within the file. The final results are then returned as the selected file type.
Perform an allignment of two sequences to see how they compare. This tool takes in two fna or fastq files containing genetic sequences and aligns them using the Needleman-Wunsch algorithm. The final results are then returned as the selected file type.
Identify variable regions in amplicon sequences, pieces of DNA/RNA that is the source of amplification or replication events. This tool takes in a fna or fastq file containing genetic sequences and plots the variance regions. The final results are then returned as the selected file type.
Assemble sequence fragments into a complete genome. This tool takes in a fna or fastq file containing genetic fragments (reads) and assembles them using a de Bruijn graph and eulerian cycle algorithm. The final results are then returned as the selected file type.
Construct a phylogeny tree from a series of genetic sequences. This tool takes in a fna or fastq file containing genetic sequences and implements the Nei-Saitou neighbor-joining algorithm for phylogeny construction. The final results are then returned as the selected file type.
